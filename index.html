Tatami

Tatami is tool for making folder structure

npm install -g tatami

Getting started

Create Tatamifile

tatami init

Run tatami

tatami run

Rollback

tatami rollback

Output Path

tatami path

Learn Tatami grammer in 30 seconds

1. Tatami parses indent, make folder structure.

2. In case of "Directory", after directory name, attach "/".

3. When name has extension or no extension, tatami recognizes "File".

4. "Comment" is allowed only oneline comment, begin "//"
